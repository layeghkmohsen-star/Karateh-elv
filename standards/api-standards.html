🛢️ API Standards

Overview of key American Petroleum Institute (API) standards for the oil and gas industry, including equipment design, safety, and quality management.

🔹 What is API?

The American Petroleum Institute (API) is a trade association that represents the oil and gas industry. API develops standards for equipment, materials, and procedures used in exploration, production, refining, and transportation of oil and gas. API standards are widely adopted globally to ensure safety, reliability, and environmental protection.

🔹 Key API Standards

  • API 551 – Process Measurement (Instrumentation)
  • API 552 – Control Valve Sizing and Selection
  • API 553 – Control Valve and Actuator Installation
  • API 554 – Process Control Systems
  • API 555 – Process Analyzers
  • API 560 – Fired Heaters for General Refinery Service
  • API 570 – Piping Inspection Code
  • API 650 – Welded Tanks for Oil Storage
  • API 653 – Tank Inspection, Repair, Alteration, and Reconstruction
  • API RP 500 – Classification of Locations for Electrical Installations
  • API RP 505 – Electrical Installations in Class I, Zone 0/1/2 Locations

🔹 API Standards by Discipline

  • Instrumentation: API 551, API 552, API 553
  • Electrical: API RP 500, API RP 505
  • Quality: API Q1 – Quality Management for Oil and Gas
  • Safety: API RP 75 – Safety and Environmental Management

🔹 Example: API 551 for Instrumentation Design

An instrumentation engineer uses API 551 to select and specify pressure, temperature, flow, and level instruments for an oil refinery. The standard provides guidelines for instrument selection, accuracy requirements, and installation practices. Compliance with API 551 ensures reliable measurement and process safety.
