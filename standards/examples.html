📝 Standards Examples

Practical projects and examples to apply engineering standards knowledge. Includes complete descriptions and learning outcomes.

🔹 Example 1: IEC 60364 Application in a Facility

Description: Design of an electrical installation for a new industrial facility following IEC 60364. Includes load estimation, cable sizing, protection coordination, and earthing system design.

  • Standard: IEC 60364 (Electrical Installations)
  • Scope: Entire facility, 400V distribution
  • Key skills: Load calculation, cable sizing, protection coordination, earthing design

🔹 Example 2: NEC Compliance for Motor Control Panel

Description: Design of a motor control panel that complies with NEC Article 430 (Motor Circuits) and Article 250 (Grounding). Includes proper overcurrent protection, disconnecting means, and conductor sizing.

  • Standard: NEC (NFPA 70)
  • Scope: Motor control panel for 3-phase induction motors
  • Key skills: NEC article application, overcurrent protection, grounding, working clearances

🔹 Example 3: ISO 9001 Documented QMS

Description: Development of a Quality Management System (QMS) for an engineering firm to achieve ISO 9001 certification. Includes quality policy, procedures, work instructions, and records.

  • Standard: ISO 9001:2015
  • Scope: Engineering design and consulting services
  • Key skills: Document control, process mapping, internal audit, corrective action

🔹 Example 4: API 551 Instrumentation Specification

Description: Using API 551 to specify pressure, temperature, flow, and level instruments for a refinery process unit. Includes instrument selection, accuracy requirements, and installation practices.

  • Standard: API 551 (Process Measurement)
  • Scope: Refinery distillation unit
  • Key skills: Instrument selection, accuracy classification, installation guidelines
