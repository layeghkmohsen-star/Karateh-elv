📈 Statistical Process Control (SPC)

Statistical methods for monitoring and controlling process quality, including control charts, process capability, and variation analysis.

🔹 What is Statistical Process Control (SPC)?

SPC is a method of quality control that uses statistical techniques to monitor and control processes. It helps distinguish between common cause variation (inherent to the process) and special cause variation (assignable to specific factors). SPC enables process stability and capability assessment.

🔹 Key SPC Tools

  • Control Charts – X-bar, R, p, np, c, u charts
  • Process Capability Analysis – Cp, Cpk, Pp, Ppk indices
  • Run Charts – Plotting data over time to identify trends
  • Pareto Analysis – Identifying the most significant factors
  • Cause-and-Effect Diagrams – Fishbone or Ishikawa diagrams

🔹 Key SPC Formulas

Cp = (USL - LSL) / (6 × σ)
Cpk = min[(USL - μ) / (3 × σ), (μ - LSL) / (3 × σ)]
UCL = μ + 3 × σ  |  LCL = μ - 3 × σ

Where USL = Upper Specification Limit, LSL = Lower Specification Limit, μ = Process Mean, σ = Process Standard Deviation.

🔹 Example: Control Chart for Machine Shop

A machine shop uses X-bar and R charts to monitor the diameter of machined parts. Samples of 5 parts are taken every hour. The X-bar chart tracks the average diameter, while the R chart tracks the range (variation) within samples. Points outside the control limits trigger investigation and corrective action.
