🔄 VFD Calculations

Calculate motor RPM from frequency and number of poles, and vice versa. Includes reference table for common motor configurations.

🔹 Frequency → RPM

Motor Speed: 1500.00 RPM
RPM = (120 × Frequency) / Poles

🔹 RPM → Frequency

Frequency: 50.00 Hz
Frequency = (RPM × Poles) / 120
