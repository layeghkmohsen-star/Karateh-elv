📝 PLC Examples & Exercises

Practical programming examples and exercises for PLC, HMI, SCADA, and industrial networks to strengthen your skills.

🔹 Example 1: Motor Start/Stop with Seal-in Circuit

Description: A classic motor control circuit using a PLC. The motor starts when the Start button is pressed and continues running (seal-in) until the Stop button is pressed.

Inputs:

  • I0.0 – Start Push Button (NO)
  • I0.1 – Stop Push Button (NC)

Outputs:

  • Q0.0 – Motor Contactor
// Ladder Logic
// Network 1: Seal-in circuit
|----[I0.0]----[I0.1]----(Q0.0)----|
|----[Q0.0]----| |

Learning Outcome: Understand how to implement a seal-in (latching) circuit in ladder logic.

🔹 Example 2: Timer – Delay ON (TON)

Description: A motor starts 5 seconds after the Start button is pressed. The TON timer provides the delay.

  • Inputs: I0.0 – Start Button
  • Outputs: Q0.0 – Motor
  • Timer: TON (Preset = 5s, Time Base = 100ms → Preset Value = 50)
// Ladder Logic
|----[I0.0]----[TON]----(Q0.0)----|
// TON: PT=50, ET=...

Learning Outcome: Learn how to use TON timers for delay-on applications.

🔹 Example 3: Counter – Count Products

Description: A conveyor system counts products passing a sensor. After 100 products, a signal is sent to a packing station.

  • Inputs: I0.0 – Product Sensor (pulse)
  • Outputs: Q0.0 – Packing Station Signal
  • Counter: CTU (Preset = 100)
// Ladder Logic
|----[I0.0]----[CTU: PV=100]----(Q0.0)----|
// Reset using I0.1

Learning Outcome: Understand how to use CTU counters for counting applications.

🔹 Example 4: Analog Scaling – 4-20mA to Temperature

Description: A temperature transmitter outputs 4-20mA corresponding to 0-100°C. Scale the analog input to display the actual temperature.

  • Input: AIW0 – 4-20mA (raw value 0-27648 for 0-20mA)
  • Output: MW10 – Temperature in °C (scaled 0-100)
// Structured Text (ST)
Raw := AIW0; // 0-27648
Temp := (Raw - 5530) * 100 / 22118; // 5530 = 4mA, 22118 = 16mA range
MW10 := Temp;

Learning Outcome: Practice analog scaling and conversion using ST.
